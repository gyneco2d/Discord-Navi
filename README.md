# Discord-Navi
A discord bot by gyneco2d using discordrb library.
It's still under development.

### implemented
- Notification when joining a voice channel.

# Usage
```
$ git clone https://github.com/gyneco2d/Discord-Navi.git
$ cd Discord-Navi
$ bundle install
$ ruby run.rb
```
